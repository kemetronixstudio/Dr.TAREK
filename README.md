# KG English Quiz App (Flat Files Version)

This version is prepared so you can upload to GitHub easily without subfolders.
All project files are in the root.

## Included files
- index.html
- kg1.html
- kg2.html
- certificate.html
- style.css
- script.js
- certificate.js
- SVG image files in the root (no folders)
- README.md
- LICENSE
- .gitignore

## Features
- English + Arabic interface
- English quiz content for KG1 and KG2
- Old quiz revised and organized into the app
- Level 1 = 10 questions
- Level 2 = 20 questions
- Level 3 = 30 questions
- Auto next after answer or manual next button
- Mobile, tablet, and desktop friendly
- Kid-friendly design
- Progress saved in localStorage
- Real certificate page
- PDF download and sharing

## Upload to GitHub
1. Create a new repository.
2. Drag all files from this ZIP into GitHub upload.
3. Commit the files.

## Deploy to Vercel
1. Import the GitHub repository in Vercel.
2. Choose the repository.
3. Deploy as a static website.
